
# rtvrrr

<!-- badges: start -->
<!-- badges: end -->

The rtvrrr package provides tools for estimating and forecasting (time-varying)
reduced rank regression models under restrictions.

## Installation

You can install the development version of rtvrrr from [GitHub](https://github.com/) with:

``` r
# install.packages("pak")
pak::pak("sebvellu/rtvrrr")
```
